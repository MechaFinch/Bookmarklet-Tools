Bookmarklet Tools

This site has a bunch of bookmarklets which range from 'tools' to random stuff.
Bookmarklets are links, usually put in a bookmark, that contain Javascript instead of a normal link. They can do anything you can do with javascript.
More will be coming when I make them, so keep a lookout for it.

To use them, drag them into the bookmarks bar and click it to run them. If you need more information, click the 'help' button here.
